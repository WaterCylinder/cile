## Scripts 脚本文件

### Core 核心类

游戏的核心类，用于掌控游戏运行的核心逻辑

* [Game](../Scripts/Core/Game.cs)

    游戏流程类

* [Global](../Scripts/Core/Global.cs)

    全局类，用于存储全局变量

### Manager 管理器类

1. 核心管理器，在游戏运行开始时加载（在GameManager里加载，而GameManager在设置里自动加载）

* [CardManager](../Scripts/Manager/CardManager.cs)
* [SceneManager](../Scripts/Manager/SceneManager.cs) 
* [InputManager](../Scripts/Manager/InputManager.cs)

2. 资源加载管理器，在游戏运行过程中加载

* [AssetManager](../Scripts/Manager/AssetManager.cs)
* [EntityManager](../Scripts/Manager/EntityManager.cs)

### GameSystem 游戏系统类

* [RoundCricle](../Scripts/Core/GameSystem/RoundCricle.cs)

    回合循环类，用于控制回合循环

### DataFormat 标准数据格式类

    定义了标准的数据格式，枚举等。

### Effect 效果类

    效果类，运行效果类的run方法时，先运行Conditon，Condition返回真时运行Behavior。包含了各种可组合的Condioon类和Behacior类，在编辑器里进行拼接组合。

### Object 实体类

    可实例化的实体

### Resource 资源数据类

    用于存储数据的定义类

### Test 测试类

    用于测试

### UI 界面类

    UI界面相关

### Utils 工具类

* [Coroutine](../Scripts/Utils/Coroutine.cs)

    携程类，用于提供类似于UnityCoroutine的携程功能，多线程功能。

* [Tools](../Scripts/Utils/Tools.cs)

    工具类，提供一些常用的工具函数。
